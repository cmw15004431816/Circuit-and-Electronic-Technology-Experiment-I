电路与电子技术实验报告

实验3—非线性元件的伏安测量法

一、实验目的和要求

实验目的

  1. 学习Multisim仿真软件在电路中的应用；
  2. 学习示波器的正确使用方法；
  3. 学习非线性元件伏安特性的示波器观察方法；
  4. 了解非线性元件的伏安特性；
  5. 学习非线性元件伏安特性的测试方法；
  6. 掌握绘制曲线的方法。

实验要求

  1. 请不要带食物进入实验室，更不允许在实验室用餐；
  2. 请勿大声喧哗，不要随意走动，不要私自更换实验设备；
  3. 请听从实验指导老师的安排，独立完成实验；
  4. 实验完毕请关闭电源，万用电表用完后关闭电源归还，并摆放整齐；整理实验桌面，保持实验室整洁；
  5. 请注意用电安全，包括人身安全和设备安全；
  6. 文明实验。

二、实验内容和原理

实验内容

  1. Multisim的应用；
  2. 示波器和信号发生器的使用；
  3. 用示波器观察二极管和稳压管的伏安特性曲线，并记录波形；
  4. 用伏安测量法绘制电路元件的特性曲线。

实验原理

  1. Multisim的基础操作；
  2. 示波器和信号发生器的使用方法；
  3. 普通晶体二极管的特性；
  4. 稳压二极管的特性。

三、主要仪器设备

实验板、直流电源、直流电压表、电流表、数字万用表、示波器、信号发生器、晶体二极管、稳压二极管、电阻、多通道隔离放大器、导线、电容（可选）。

四、操作方法和实验步骤

  1. 用示波器分别观察普通二极管和稳压管的伏安特性并记录两种元件的伏安特性曲线；
  2. 分别测定普通二极管有电阻R和无电阻R时的伏安特性；
  3. 测量稳压二极管有电阻R的伏安特性；
  4. 分别绘制二种元件的伏安特性曲线。

按照指导，合理连接电路即可。

六、实验结果与分析

实验结果

实验结果的分析

在图1中，我们能够非常清晰的看到二极管的单向导电性这一性质，接正向电压时导通，接反向电压时阻断。图2中的二极管的伏安特性曲线也可以很好的反映这一点。

图3与图1类似，但又有区别。最明显的区别在于稳压管在反向电压较大时会被击穿，表现为反向导通。而对于普通二极管，在我们实验的电压范围内，它是无法被反向击穿的。图4的伏安特性曲线也说明了这一点。我们只需要对比图2和图4的图像的最左侧，即可发现普通二极管在实验过程中没有被击穿，而稳压管在实验过程中确实有被击穿。

对比图5和图6，你不会发现什么区别。事实上也确实如此。因为是否接入保护电阻R实际上完全不会影响到普通二极管的伏安特性。实际上，如果去掉稳压二极管的保护电阻R，理论上你也可以得到几乎相同的曲线。但是由于我们的实验器材的功率，电流等都有限制，对于稳压管，我们必须加保护电阻以确保不会过流，保证安全，所以我们没有做去掉稳压管的保护电阻R进行测量的实验。

对比图5、6和图2，你会发现图线的形状大致是相同的，这也印证了我们的实验得到了正确的结果。同理，图7与图4的图线形状也大致相同。
